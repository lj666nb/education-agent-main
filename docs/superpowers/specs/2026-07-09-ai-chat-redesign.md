# AI 对话界面全面重设计

**日期**: 2026-07-09
**对标**: https://qwen.ai/home
**状态**: 设计已确认

---

## 1. 设计目标

将现有 AI 对话界面改造为 Qwen 风格的现代简洁设计，保持浅色主题，重点提升质感和布局。

## 2. 设计决策汇总

| 决策项 | 选择 |
|--------|------|
| 主题 | 保持浅色，优化质感 |
| 改造范围 | 全面重设计（侧边栏 + 头部 + 消息区 + 输入区 + 空状态） |
| 输入区风格 | 居中大输入卡片 + 内嵌功能按钮 |
| 消息气泡操作栏 | 纯图标按钮：重新生成 / 朗读 / 复制 |
| 图标库 | **Lucide React** (`lucide-react`) |
| 功能标签 | 模型选择 / 深度思考 / 联网搜索 / 选择项目 / PPT 生成 |

## 4. 组件设计详情

### 4.1 顶栏 (Header)
- 透明/毛玻璃态背景（`backdrop-filter: blur(8px)`）
- 左侧：侧边栏折叠按钮（Lucide `PanelLeft` 图标）
- 居中：项目 Logo/标题（可选）
- 右侧：项目选择、Diagram、Code 切换按钮 + 新建对话按钮
- 高度：56px
- 底部淡分隔线

### 4.2 侧边栏 (Sidebar)
- 可完全折叠为图标模式（仅显示图标）
- 顶部：搜索框（带搜索图标）
- 收藏区：置顶显示收藏的对话（⭐ 星标）
- 历史区：按时间分组（今天 / 昨天 / 更早）
- 列表项：标题 + 模型标签 + 时间，hover 显示操作（星标/删除）
- 底部：新建对话按钮
- 展开宽度 260px，折叠宽度 56px，带过渡动画

### 4.3 消息气泡

#### AI 消息
- 白色卡片背景 (`#FFFFFF`)
- 边框：`1px solid #F0F0F0`
- 阴影：`0 1px 3px rgba(0,0,0,0.04)`
- 圆角：`12px 12px 12px 4px`（左下角小圆角）
- 无头像
- 操作栏（始终可见，消息底部）：
  - 重新生成（Lucide `RefreshCw`）
  - 朗读（Lucide `Volume2`）
  - 复制（Lucide `Copy`）
- 宽度：max-width 85%
- 内容：Markdown 渲染（代码高亮、表格、图片等）
- 深度思考：折叠式 `<details>`，思考中显示加载动画

#### 用户消息
- 背景：品牌蓝微渐变 `linear-gradient(135deg, #0284C7, #0369A1)`
- 文字：白色
- 圆角：`16px 16px 4px 16px`
- 宽度：max-width 70%
- 靠右对齐
- 间距：消息间距增大至 24px

### 4.4 输入区（居中大卡片）

- 外层容器居中，`max-width: 768px`，左右自动 margin
- 白色卡片背景，大圆角 `16px`，柔和阴影
- 与消息区间距 20px

#### 4.4.1 建议问题区
- 输入框上方显示 3-5 个建议问题标签
- 圆角药丸样式，浅灰背景
- 点击直接填入输入框并发送

#### 4.4.2 输入框主体
- 多行 textarea，自动伸缩（max-height 200px）
- 左侧附件上传按钮（Lucide `Paperclip`）
- 右侧发送按钮（Lucide `ArrowUp`），圆形，品牌色
- 发送按钮无文字，仅图标
- placeholder: "输入消息..."
- 支持 Enter 发送，Shift+Enter 换行
- 支持粘贴文件

#### 4.4.3 底部功能标签栏
- 水平排列的功能切换标签
- 左对齐
- 每个标签：小圆角按钮，切换开关状态
- 激活态：品牌色背景 + 白色文字
- 非激活态：浅灰背景 + 灰色文字
- 标签列表：
  1. **模型选择**（下拉）：显示当前模型名 + 下拉箭头（Lucide `ChevronDown`）
  2. **深度思考**（切换）：图标 Lucide `Brain`
  3. **联网搜索**（切换）：图标 Lucide `Globe`
  4. **选择项目**（下拉）：图标 Lucide `Folder`
  5. **PPT 生成**（切换）：图标 Lucide `Presentation`

### 4.5 空状态

- 居中大标题"开始新对话"（字体 1.75rem，品牌色深色）
- 副标题：简短描述
- 3-4 个功能引导卡片（白色，圆角，阴影），每个带图标 + 标题 + 描述：
  - 文件上传（Lucide `FileUp`）
  - 代码运行（Lucide `Play`）
  - 图表生成（Lucide `BarChart3`）
  - 思维导图（Lucide `GitBranch`）
- 底部：2-3 个快捷建议标签

### 4.6 加载/生成中状态
- 生成中：AI 消息区显示跳动的圆点动画（非普通 spinner）
- 停止生成按钮合并到发送按钮位置（生成中时发送按钮变为停止按钮）

### 4.7 动画规范
- 消息出现：`fade-in` + 轻微上滑（`translateY(8px) → 0`），300ms
- 侧边栏折叠：宽度过渡 250ms `cubic-bezier(0.4, 0, 0.2, 1)`
- 按钮 hover：背景色微变 150ms
- 建议问题标签 hover：轻微上浮 + 阴影增强
- 输入卡片聚焦：边框变为品牌色，阴影增强
